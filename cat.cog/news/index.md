# Changelog

## cat.cog 0.1.0

- Initial release of the R interface to the Python `catcog` package.
- Specialized tools for cognitive assessment scoring, including CERAD
  Constructional Praxis test evaluation for dementia research.
- Exports
  [`cerad_drawn_score()`](https://christophersoria.com/cat-llm/cat.cog/reference/cerad_drawn_score.md)
  for scoring hand-drawn shapes via LLM vision models.
