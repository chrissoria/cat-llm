# Re-exported functions from cat.stack

The `cat.llm` meta-package re-exports the core functions from
`cat.stack` so they are available immediately after
[`library(cat.llm)`](https://christophersoria.com/cat-llm/cat.llm/).
Behavior and arguments are identical to the originals — the links below
point at the canonical documentation in `cat.stack`.

## Details

- [`classify`](https://christophersoria.com/cat-llm/cat.stack/reference/classify.html):

  Classify text, images, or PDFs into predefined categories.

- [`extract`](https://christophersoria.com/cat-llm/cat.stack/reference/extract.html):

  Discover and extract categories from data when no scheme is
  predefined.

- [`explore`](https://christophersoria.com/cat-llm/cat.stack/reference/explore.html):

  Raw category extraction for saturation analysis.

- [`summarize`](https://christophersoria.com/cat-llm/cat.stack/reference/summarize.html):

  Summarize text, images, or PDFs in several formats.

- [`prompt_tune`](https://christophersoria.com/cat-llm/cat.stack/reference/prompt_tune.html):

  Interactive, human-in-the-loop tuning of the classification system
  prompt.

- [`install_cat_stack`](https://christophersoria.com/cat-llm/cat.stack/reference/install_cat_stack.html):

  One-time installer for the Python `catstack` backend.

- [`ensure_ollama_running`](https://christophersoria.com/cat-llm/cat.stack/reference/ensure_ollama_running.html):

  Check (and optionally start) a local Ollama server before running
  local-model classification.

- [`list_ollama_models`](https://christophersoria.com/cat-llm/cat.stack/reference/list_ollama_models.html):

  List locally installed Ollama models.

- [`check_ollama_model`](https://christophersoria.com/cat-llm/cat.stack/reference/check_ollama_model.html):

  Check whether a specific Ollama model is installed locally.

- [`pull_ollama_model`](https://christophersoria.com/cat-llm/cat.stack/reference/pull_ollama_model.html):

  Download an Ollama model to the local installation.
