# Changelog

## cat.ademic 0.1.0

- Initial release of the R interface to the Python `catademic` package.
- Domain wrapper around `cat.stack` that adds journal and topic sourcing
  parameters (via OpenAlex) for academic literature analysis.
- Exports
  [`classify()`](https://christophersoria.com/cat-llm/cat.ademic/reference/classify.md),
  [`extract()`](https://christophersoria.com/cat-llm/cat.ademic/reference/extract.md),
  [`explore()`](https://christophersoria.com/cat-llm/cat.ademic/reference/explore.md),
  and
  [`summarize()`](https://christophersoria.com/cat-llm/cat.ademic/reference/summarize.md).
