# Changelog

## cat.vader 0.1.0

- Initial release of the R interface to the Python `catvader` package.
- Domain wrapper around `cat.stack` adapted for social media data.
- Connects to the Threads API (and other social platform connectors) to
  pull post archives, classify them, and return enriched datasets with
  engagement metrics.
- Exports
  [`classify()`](https://christophersoria.com/cat-llm/cat.vader/reference/classify.md),
  [`extract()`](https://christophersoria.com/cat-llm/cat.vader/reference/extract.md),
  and
  [`explore()`](https://christophersoria.com/cat-llm/cat.vader/reference/explore.md).
