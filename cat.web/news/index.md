# Changelog

## cat.web 0.1.0

- Initial release of the R interface to the Python `catweb` package.
- Domain wrapper around `cat.stack` that adds automatic URL fetching and
  web-context prompt injection (source domain, content type, metadata).
- Exports
  [`classify()`](https://christophersoria.com/cat-llm/cat.web/reference/classify.md),
  [`extract()`](https://christophersoria.com/cat-llm/cat.web/reference/extract.md),
  [`explore()`](https://christophersoria.com/cat-llm/cat.web/reference/explore.md),
  and
  [`summarize()`](https://christophersoria.com/cat-llm/cat.web/reference/summarize.md).
